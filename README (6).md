
# Spotify Global Listener & Content Performance Dashboard

## Overview
This Streamlit dashboard provides a **stakeholder-level view** of Spotify's global content and listener performance.

## Key Questions Answered
- Which artists and genres drive the most engagement?
- How does listening vary across regions?
- How do Free and Premium users differ?
- Where are growth opportunities and risks?

## Features
- Global filters (region, genre, subscription, month)
- Visual-level filters for deeper exploration
- KPI cards for quick executive insights
- Clear highs & lows to reflect realistic performance

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Theme
Spotify-inspired dark UI with clean, executive-ready visuals.
